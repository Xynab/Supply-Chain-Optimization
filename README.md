# Supply-Chain-Optimization
Supply Chain Optimization using Python, SQL, Forecasting and Streamlit Dashboard
This project implements an end-to-end supply chain optimization system using Python and SQL.

Features
1.Sales data analysis using SQL
2.Demand forecasting using Prophet
3.Inventory optimization using EOQ and Reorder Point logic
4.Automated reorder decision system
5.Interactive dashboard using Streamlit

Tech Stack
1.Python
2.SQL (SQLite)
3.Pandas
4.Prophet
5.Streamlit

Dataset:
Kaggle Sales Forecasting Dataset

How to Run:
streamlit run app.py
